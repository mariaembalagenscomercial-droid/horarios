# Widget Status Loja - Maria Embalagens

Servidor que retorna a imagem correta (ABERTA/FECHADA) baseado no horário de Brasília.

## Uso

```html
<img src="https://maria-widget.onrender.com/status-widget.png" alt="Status da Loja" style="max-width: 100%; height: auto;">
```

## Horários

| Dia | Horário |
|-----|---------|
| Segunda | 22:00 — 23:59 |
| Terça | 00:00 — 10:00 |
| Quarta | 05:00 — 10:00 \| 22:00 — 23:59 |
| Quinta | 00:00 — 10:00 |
| Sexta | 05:00 — 10:00 \| 22:00 — 23:59 |
| Sábado | 00:00 — 10:00 |
| Domingo | FECHADO |
